# Environment Overview

This project simulates a real-world Security Operations Center (SOC)
environment using Wazuh SIEM to monitor and detect attacks across
multiple endpoints.

## Lab Components
- Wazuh Manager & Dashboard
- Windows 10 Endpoints (Aditya, Jamod)
- Ubuntu Linux Endpoint (Devang)
- Kali Linux (Attacker)
- Centralized log collection & alerting

## Objective
To design and validate detection use cases that reflect real enterprise
SOC monitoring, alerting, and investigation workflows.
