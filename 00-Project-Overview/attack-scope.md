# Attack Scope

All attacks in this project were performed in a controlled lab
environment owned by the author.

## In-Scope Attacks
- SSH Brute Force (Linux)
- RDP Brute Force (Windows)
- Failed Login Attempts
- Network Reconnaissance (Nmap)

## Out-of-Scope
- No production systems
- No unauthorized networks
- No real user data

This scope aligns with ethical security testing and blue-team validation.
